Gebruiksaanwijzing 
------------------

#. ga naar de `releases <https://github.com/axd1967/sl-l-textielinstituut-gent/releases>`_ pagina
#. download het zip bestand
#. in Stellarium importeer je dit .zip bestand via F4 > Landscape > Add/remove landscape
